XList Database

This is a database of data services ready to be used on an xlist server. The database has been classified according to the class used in its configuration.

Warning
 Each list provider has its own usage policy, check its website before using it. In order to use Google Safe Browsing API, you will need to obtain an API key.

  • sblookup data services (Google safe browsing)

  • file data services

  • dnsxl data services

In order to use the data services of the class file it is necessary to download and keep the data synchronized. For this purpose the xlget tool must be used and this tool needs the configuration of the sources.
